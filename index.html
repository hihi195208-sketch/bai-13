Luyện tập

Bài 1

Câu hỏi: “Ngôn ngữ định dạng CSS chính là ngôn ngữ HTML” là đúng hay sai?

Trả lời: Sai. Ngôn ngữ định dạng CSS (Cascading Style Sheets) không phải là ngôn ngữ HTML (HyperText Markup Language). HTML được dùng để tạo cấu trúc nội dung bằng các thẻ, còn CSS dùng để điều chỉnh cách hiển thị các phần tử HTML như màu sắc, kích thước chữ, khoảng cách, bố cục… CSS là một ngôn ngữ độc lập và hoạt động cùng với HTML để tạo giao diện hoàn chỉnh.

Bài 2

Câu hỏi: “Có thể chỉ cần thay đổi thông tin của tệp CSS sẽ làm thay đổi định dạng của nhiều trang web hoặc cả một website” đúng hay sai?

Trả lời: Đúng. Khi nhiều trang web hoặc nhiều trang trong cùng một website cùng sử dụng một tệp CSS, việc thay đổi thuộc tính trong tệp CSS sẽ lập tức làm thay đổi giao diện của tất cả các trang đang liên kết đến tệp đó. Điều này cho phép việc chỉnh sửa giao diện trở nên nhanh chóng, thống nhất và không cần chỉnh từng trang.

Vận dụng

Sự giống nhau và tương thích của Style Sheet trong các phần mềm soạn thảo văn bản với CSS của trang web:

  • Cấu trúc và cú pháp: Cả Style Sheet và CSS đều tuân theo một quy tắc định dạng. Trong phần mềm soạn thảo văn bản, Style Sheet định nghĩa font chữ, kích thước, căn lề… được áp dụng cho đoạn văn; trong CSS, các thuộc tính cũng được viết theo dạng "tên thuộc tính: giá trị".
  • Tính tái sử dụng: CSS cho phép định dạng một lần và áp dụng cho nhiều phần tử HTML. Tương tự, Style Sheet trong Word hay LibreOffice cho phép áp dụng cùng một kiểu định dạng cho nhiều đoạn văn khác nhau.
  • Khuôn mẫu định dạng: Cả hai đều cho phép tạo khuôn định dạng cho tiêu đề, đoạn văn, danh sách… giúp văn bản hoặc trang web có phong cách thống nhất.
  • Kiểm soát phong cách: Cả CSS và Style Sheet đều cho phép tùy chỉnh màu sắc, font chữ, khoảng cách, độ đậm, đường viền… giúp người thiết kế chủ động tạo bố cục và thẩm mỹ mong muốn.
